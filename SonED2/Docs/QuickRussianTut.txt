
 Быстрый русский туториал по SonED2 by Vladikcomper
 ---------------------------------------------------
 
 Для начала, нажмите File>Open Project и выберите уровень, который
 вас интересует.
 
 Переключение между режимами редактирования производится с помощью
 клавиш:
 
 I - Редактор Тайлов
 O - Редактор объектов
 L - Редактор уровня
 P - Редактор фона уровня
 
 И еще: почаще сохраняйтесь в первое время работы с программой (F3),
 так как там нет команды Undo.
 
 Редактор уровня
 ----------------
 
 Нажимайте A или Z, чтобы сменить вставляемый блок, его id будет 
 меняться при нажатии этоих клавиш (см. строчку "CURRENT CHUNK").
 Нажмите левую кнопку мыши, чтобы вставить выбранный блок в уровень.
 
 
 Редактор Объектов
 ------------------
 
 Объекты можно таскать с помощью мыши.
 
 Двойной щелчок левой кнопкой мыши - Вставить новый объект/Выбрать тип объекта
 Двойной щелчок правой кнопкой мыши - Удалить объект
 
 При наведении курсора на объект, в верхнем левом углу отображаются
 его свойства. С помощью клавиш S/X, D/C вы можете изменять 
 свойства объекта, а клавиши A/Z подобно двойному щелчку правой кнопкой,
 позволяют сменить тип объекта.
 
 
 Редактор Тайлов
 ----------------
 
 Выберите нужный блок (Chunk) из списка слева. Его изображение появится на экране
 в нижнем левом углу. Там вы сможете выбрать блоки 16х16, правой (!) кнопкой мыши.
 Левая кнопка - рисовать блок 16х16. При двойном щелчке правой кнопкой изображение
 блока появляется в верхней части экрана, разбитое на четыре блока 8х8. При двой-
 ном щелчке правой кнопкой мыши блок 8х8 (тайл) появится (ну, увидите где), после
 чего на нем можно рисовать. Цвета выбираются в палитре. На один блок приходится
 1 строка палитры - 16 цветов.
